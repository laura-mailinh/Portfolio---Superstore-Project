# Portfolio---Superstore-Project
**Business Performance Analysis & Forecasting Dashboard**

**Project Overview**
This project analyzes business performance using the Superstore dataset, covering:

Sales Performance
Customer Behavior
Profitability Analysis
Regional Performance
Sales Forecasting

**Tools Used**
Python (Pandas, Prophet, Statsmodels)
Tableau
Data Visualization

**Key Insights**
**Sales Performance**
Sales increased by 20.36%
High-value customers generated 64.83% revenue

**Customer Behavior**
Repeat customer rate reached 98.49%
Repeat customers contributed 77.68% revenue

**Profitability**
Profit margin declined by 5.09%
Loss-making products increased 29.98%
Discounts above 30% created losses

**Regional Performance**
West region most profitable
Central region underperformed

**Forecast**
Forecast accuracy: 84.13%
Expected growth: 25.09%
